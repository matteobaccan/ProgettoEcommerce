# ProgettoEcommerce

Progetto realizzato per il corso WebDesign2: esempio di creazione e pubblicazione di un semplice sito ecommerce frontend, pubblicato su Netlify e con alcune pagine generate con l'ausilio di strumenti di AI (es. ChatGPT) durante la fase di prototipazione.

Sito demo: https://progettoecommerce.netlify.app/

[![Netlify Status](https://api.netlify.com/api/v1/badges/7cc334cc-6945-41dc-b9c9-0355df57ec6d/deploy-status)](https://app.netlify.com/sites/progettoecommerce/deploys)

## Descrizione
Questo repository contiene il codice sorgente del frontend di un esempio di e-commerce didattico. L'obiettivo è mostrare il flusso di sviluppo di una semplice interfaccia, la gestione delle pagine prodotto e il deploy su Netlify.

## Funzionalità principali
- Pagine prodotto di esempio
- Layout responsive per dispositivi mobili e desktop
- Deploy continuo su Netlify
- Esempi di contenuti e testi creati come supporto alla progettazione delle pagine

## Tecnologie utilizzate
- HTML5
- CSS3
- JavaScript (vanilla)
- Netlify per il deploy

## Struttura del progetto
- / (root) — file di configurazione e il file index.html principale
- /css — fogli di stile
- /js — script lato client
- /assets — immagini e risorse statiche

Adatta questa sezione alla struttura reale del repository aggiungendo i nomi delle cartelle e dei file principali se necessario.

## Esecuzione in locale
1. Clona il repository:
   git clone https://github.com/matteobaccan/ProgettoEcommerce.git
2. Apri index.html nel browser oppure usa un server locale (es. Live Server in VS Code) per testare il sito.

(Se il progetto utilizza un workflow con npm / build step, aggiungere i comandi corretti: npm install, npm run dev / build, ecc.)

## Deploy su Netlify
Il progetto è pubblicato su Netlify. Per aggiornare il sito:
- Collega il repository a Netlify (se non già fatto)
- Effettua push sui branch configurati per il deploy
- Netlify eseguirà automaticamente il deploy alla ricezione delle modifiche
